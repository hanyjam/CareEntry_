# CareEntry — Medical Web Application

A full-stack medical web application developed as a university team project (Medical Informatics, Wilhelm-Büchner University).

## About
CareEntry digitizes patient workflows in a clinical setting, enabling secure communication between patients and medical staff.

## Features
- Patient intake forms
- Lab value visualization (charts)
- Secure messaging between patients and physicians
- Protected routes / authentication
- File upload functionality

## Tech Stack
- React, JavaScript (JSX)
- Tailwind CSS
- Vite
- REST APIs

## Role
Responsible for frontend implementation including component architecture, lab value charts, and REST API integration.
